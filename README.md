# fn-apps

飞牛OS应用 仓库。

## 模块速览
| 应用名称 | 版本 | 平台 | 描述 |
|----------|------|------|------|
| fn-codeserver | v1.0.3 | all | code-server 是 VS Code 的在线版本，允许您通过浏览器进行代码编辑和开发。 |
| fn-fail2ban | v1.0.0 | all | fail2ban 是一个开源的入侵防御工具，用于保护 Linux 服务器免受暴力破解攻击。 它通过监控日志文件，检测可疑的登录尝试，并自动封禁恶意 IP 地址，从而增强系统的安全性。 |
| fn-kodi | v1.0.3 | x86 | Kodi 是一个免费且开源的媒体播放器软件，用于播放视频、音频和图像。 |
| fn-msedge | v1.0.4 | x86 | Microsoft Edge 是一款由 Microsoft 开发、基于 Chromium 的跨平台网页浏览器。 |
| fn-ntfs3 | v1.0.3 | all | ntfs3 驱动是一个用于 Linux 内核的开源 NTFS 文件系统驱动程序，旨在提供对 NTFS 格式化分区的高性能读写支持。 |
| fn-open-vm-tools | v1.0.6 | all | Open-VM-Tools 是 VMware Tools 的开源替代品，旨在为运行在 VMware 环境中的虚拟机提供更好的性能和用户体验。 |
| fn-qemu-ga | v1.0.6 | all | QEMU Guest Agent 是一个运行在虚拟机内部的守护进程，旨在通过与虚拟化主机的交互，执行一系列操作以增强虚拟机的管理能力。 |
| fn-scheduler | v1.1.6 | all | 轻量级的任务计划应用，支持设置定时任务以自动执行脚本或命令，同时也支持基于条件的任务触发，例如在特定时间、特定事件或特定文件变化时执行任务。 |
| fn-sshd-config | v1.0.4 | all | 设置 root 用户密码，修改 SSHD 配置. |
| fn-terminal | v1.0.7 | all | terminal (ttyd + tmux) 是一个基于 Web 的终端应用程序，允许用户通过浏览器访问和管理服务器终端会话，提供便捷的远程终端操作体验。 |
| fn-wifi-hotspot | v1.0.7 | all | 无线热点创建工具，允许用户轻松地将计算机变成一个 Wi-Fi 热点，分享网络连接给其他设备。 |
